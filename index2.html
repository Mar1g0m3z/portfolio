Bus Tracker

Tracks the location of Bus Route 70 of the Washington Metropolitan Transit Authority. We asked use an API to source real-time data of the WMT busses. I learn how to display the live location of the busses on a map

Pacmen Factory

In this exercise, I create PacMen with the click of a button and each of them will move about the browser window at their own pace when pushing the 'Start' button. In this exercise, I create a web page that produces a PacMan on the click of a button by using the array methods you've learned previously and the new techniques I have learned by manipulating the DOM. I drew the pacman as well as the background and Title used in the exercise.

Eye Movement

In this exercise I used the customizable power of HTML, JavaScript, and CSS and combined them to create some eyes that follow your cursor around. I hope to improve this project by drawing my own eyes and adding drawn eyelids that blink.
